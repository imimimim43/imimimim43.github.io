**분할 정복 알고리즘(Divide and conquer algorithm)**

-> 해결할 수 없는 문제를 작은 문제로 분할하여 문제를 해결하는 알고리즘.

 > 분할 정복 알고리즘을 사용하여 O($n^2$)를 O($n$)나 O($log_2 n$)까지는 아니더라도 O(n $log_2n$)으로 바꿀 수 있다.

<br/>

**합병 정렬(Merge Sort)**

-> O(n $log n$) 비교 기반 정렬 알고리즘. 분할 정복 알고리즘 중 하나이며 일반적인 방법으로 구현했을 때 안정 정렬에 속한다.
![합병정렬](https://search.pstatic.net/common/?src=http%3A%2F%2Fblogfiles.naver.net%2FMjAyMDA3MzBfOTAg%2FMDAxNTk2MDcxMTY3Mzkw.HBZBJDLMu2fM7CcS_jBAbfI8mmHSaMZDzGp75iKUTHog.Y5exv5QkBOV_2V1B6zORlBjX_OCPIpnptnD89FF3cr8g.PNG.wongoni%2F%25C4%25B8%25C3%25B3.PNG&type=sc960_832)

* 합병 정렬 알고리즘
```
MergeSort(A, p, q)
입력: A[p]~A[q]
출력: 정렬된 A[p]~A[q]
if( p < q ){
    k = [(p+q)/2]
    MergeSort(A, k+1, q)
    A[p]~A[k]]와 A[k+1]~A[q]를 합병.
}
```


<br/>

**퀵 정렬(Quick)**

-> 배열의 요소를 정렬하기 위한 분할 정복 알고리즘의 일종으로, n개의 원소를 정렬하는 경우 평균 시간 복잡도는 O(n $log n$) 
![퀵정렬](https://search.pstatic.net/common/?src=http%3A%2F%2Fblogfiles.naver.net%2FMjAxODA3MjJfNTAg%2FMDAxNTMyMjM3MzU2ODkx.qoPVSBaf10vs2zfMMmmZ-2HBnKQqknA6JXi5xKjtUDkg.LOrJGYT98jVEpBQYKSVeiVEv7WYndjlu47fsRLun6LEg.PNG.skyvvv624%2F%25C4%25B8%25C3%25B3.PNG&type=sc960_832)

```
QuickSort(A, left, right)
입력: 배열 A[left]~A[right]
추력: 정렬된 배열 A[left]~A[right]
if(left ,rigt){
    피봇을 A[left]~A[right] 중에서 선택하고, 피봇을 A[left]와 자리를 바꾼 후, 피봇과 배열의 각 원소를 비교하여 피봇보다 작은 숫자들은 A[left]~A[p-1]로 옮기고, 피봇보다 큰 숫자들은 A[p+1]~A[right]로 옮기며, 피봇은 A[p]에 놓는다.
    QuickSort(A, left, p-1)
    QuickSort(A, p+1, right)
}
```

## Markdown문법

* $n^2$ : 양 옆에 $를 붙이고 n^2를 넣으면 됨.

* O(n $log_2n$) : 양 옆에 $ 붙이고 nlog_2n를 넣으면 됨.

-> 대부분의 수학 기호는 양 옆에 $를 붙이면 출력됨!
